# Graphic Designer Portfolio (Flask)

موقع بورتفوليو احترافي لمصمم جرافيك باستخدام Flask + SQLite مع دعم العربية RTL وتصميم حديث ومتجاوب.

## المزايا

- واجهة زائر عامة لعرض الأعمال بتصنيفات.
- لوحة إدارة خاصة (تسجيل دخول بسيط).
- إضافة/تعديل/حذف الأعمال مع رفع الصور.
- إدارة التصنيفات (إضافة/حذف مع حماية من حذف التصنيف المرتبط بأعمال).
- تصميم عصري هادئ ومتوافق مع الشاشات الصغيرة والكبيرة.

## متطلبات التشغيل

- Python 3.9+

## التثبيت والتشغيل

1. تثبيت الحزم:

```bash
pip install -r requirements.txt
```

2. تشغيل التطبيق:

```bash
python app.py
```

3. افتح المتصفح على:

```text
http://127.0.0.1:5000
```

## بيانات دخول الإدارة (الافتراضية)

- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

يمكن تغييرها عبر متغيرات البيئة:

- `ADMIN_USERNAME`
- `ADMIN_PASSWORD`
- `FLASK_SECRET_KEY`

## مسارات مهمة

- الصفحة العامة: `/`
- تسجيل دخول الإدارة: `/admin/login`
- لوحة الإدارة: `/admin`

## ملاحظات

- الصور تُخزن داخل `static/uploads/`.
- قاعدة البيانات تُنشأ تلقائيًا باسم `portfolio.db` عند أول تشغيل.
